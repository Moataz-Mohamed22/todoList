# 📝 Todo List App

A simple and responsive Todo List application built with **HTML**, **CSS**, and **JavaScript**. It allows users to manage their daily tasks with automatic data persistence using **Local Storage**.

## ✨ Features

- Add new tasks
- Add tasks by pressing the **Enter** key
- Mark tasks as completed
- Delete tasks
- Automatically save tasks using Local Storage
- Restore saved tasks after refreshing the page
- Clean and responsive user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/todo-list.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No additional setup or installation is required.
